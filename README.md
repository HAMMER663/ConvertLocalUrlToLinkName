ConvertLocalUrlToLinkName
=========================

Will transform the internal links of type viewtopic(forum).php?f=1&amp;amp;t=1 to the links [url=viewtopic(forum).php?f=1&amp;amp;t=1]Topic(Forum) title[/url]
